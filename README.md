# 瑞吉外卖

#### 介绍
小小的一个项目演示

#### 软件架构
软件前端架构：VUE（非脚手架）,Vant组件,Element UI,Axios,CSS，JS
软件后端架构：Spring BOOT，MySQL

#### 安装教程

1.导入项目，导入数据库
2.配置好环境，点击Run
3.或者打包成jar
